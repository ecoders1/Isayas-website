# Assets

Place your images here:

- `profile.jpg` — Your professional profile photo (recommended: square, at least 400×400px)
- `project1.jpg` — E-Commerce Platform screenshot
- `project2.jpg` — Task Management App screenshot
- `project3.jpg` — Developer Portfolio screenshot

If these images are missing, the site falls back gracefully:
- Profile photo falls back to a generated avatar from ui-avatars.com
- Project images show a gradient placeholder background
